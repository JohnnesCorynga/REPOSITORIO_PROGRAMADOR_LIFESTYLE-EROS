Programador:

O programador é o profissional que cria, desenvolve e mantém diferentes tipos de softwares em sistemas amplos ou para uso em computadores pessoais. Ele utiliza diversas linguagens de programação e tecnologias para implementar soluções computacionais para resolver os problemas das pessoas e das empresas.

Isso mesmo, programador é um solucionador de problemas. Já parou para pensar como isso é gratificante?
